        launch_option()

        @type launch_option() ::
  {:use_rocksdb, boolean()}
  | {:supervisor, [Supervisor.option()]}
  | {:testing, boolean()}

        I control options for launch_min/2

        ###Options

        • :use_rocksdb - see Anoma.Node.configuration/0 for more information
        • :supervisor - This flag determine if we use a supervisor and if so what options. See Supervisor.option/0 for supervisor options
        • :testing - This flag notes if we are testing the node. This gets fed directly into the type Anoma.Node.configuration/0 for Anoma.Node.start_link/1. Please consult the Anoma.Node.configuration/0 documentation for the full effect this has on the node
